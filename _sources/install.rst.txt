Quickstart
==========

Setup
-----

To install, you need to navigate to the package directory. The package directory
is such that when you list-directory with ``ls`` you can see a ``setup.py`` file

.. code-block:: bash

    $ python -m pip install -e .

Running
-------

The ``main`` module takes command-line arguments and can be run from any directory
once it is installed.

To start a ``Listener`` instance, you need to run the main module on the specified port:

.. code-block:: bash

   $ python -m uwaPySense.main -p <PORT>

You can also specify the baud-rate with the ``--baud-rate`` flag

.. code-block:: bash

   $ python -m uwaPySense -p COM3 --baud-rate 9600

For more information see:

.. code-block:: bash

   $ python -m uwaPySense.main --help
