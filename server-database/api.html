This is an abstract base class that all implementations should derive from

class app.database.BaseBinDb¶
add_bin(bin_id, position=None, capacity=None, fill_threshold=None)¶

Adds a bin to the database

Parameters:
  • bin_id (str) – The unique bin identifier
  • position (tuple, 2) – The lat, long of the bin
  • capacity (float) – The capacity of the bin
  • fill_threshold (float) – The threshold of the bin before it is full If fill_threshold is None, but capacity > 0 then it the threshold is automatically set to the capacity
data¶

Returns all the data stored in the database

get_info_bin(bin_id, key=None)¶

Returns the bin information for a particular bin

Parameters:
  • bin_id (str) – The bin to get information for
  • key (str, default=None) –
    this will get a particular piece of
    information
    relevant to the bin. Available keys are:
    • lat (float): The latitude of the bin
    • long (float): The longitude of the bin
    • capacity (float): The capacity of the bin
    • threshold (float): The threshold for filled bins
Returns:
